# Mejora de Imágenes en Condiciones de Baja Iluminación

Este repositorio forma parte del proyecto académico de la asignatura "Visión Artificial", desarrollado dentro del Máster Universitario en Inteligencia Artificial (UNIR).

En este trabajo, abordo el reto de recuperar y realzar información visual en imágenes urbanas nocturnas mediante técnicas de procesamiento digital.  
Más allá de la mejora técnica, este proyecto refleja mi interés por construir sistemas de percepción artificial más robustos, humanos y aplicables a escenarios reales.

## Técnicas aplicadas:
- Transformaciones de intensidad (negativo, logarítmica, corrección gamma).
- Procesado de histogramas (expansión de contraste, ecualización).
- Operaciones aritméticas (suma y resta de versiones mejoradas).

## Herramientas:
- Python (Google Colab)
- OpenCV
- NumPy
- Matplotlib
